# 📊 Excel Retail & Sales Analysis Project  

## 📌 Overview  
This project was completed as part of a **Data Technician Bootcamp (Week 1 – Excel Module)** and focuses on analysing **retail and sales datasets** using core Excel tools and techniques.  

The project demonstrates the ability to clean, transform, and analyse data to uncover insights using formulas, PivotTables, and visualisations.

---

## 🧠 Key Skills Demonstrated  

### 🔢 Excel Functions & Formulas  
- `SUM` – Calculate total sales and commissions  
- `SUMIF` – Conditional aggregation of sales data  
- `AVERAGE` & `AVERAGEIF` – Identify trends and averages  
- `DATE`, `MONTH`, `YEAR` – Work with time-based data  
- `UNIQUE` – Extract distinct values from datasets  
- `VLOOKUP` – Retrieve and match related data across tables  
- `SWITCH` – Categorise sales volume into meaningful groups (High / Medium / Low)  

---

### 📂 Data Manipulation  
- Filtering data (e.g. analysing age groups and transaction details)  
- Sorting datasets (e.g. largest to smallest, restoring original order)  
- Data cleaning and preparation for analysis  

---

### 📊 Data Analysis Tools  
- Creating and customising **PivotTables** to summarise:  
  - Sales by product  
  - Sales by location (county/country)  
  - Customer demographics  

---

### 📈 Data Visualisation  
- Building charts and dashboards from sales data  
- Comparing chart types (e.g. bar vs stacked charts) to improve clarity  
- Presenting key trends and business insights visually  

---

## 📁 Project Tasks  

### ✅ Retail Dataset Analysis  
- Applied filtering and sorting techniques  
- Calculated totals and averages for commissions  
- Explored patterns in customer and transaction data  

### ✅ Pivot Table Analysis (Bike Sales)  
- Analysed customer demographics by age group and country  
- Identified most profitable segments (e.g. USA Female Adults)  
- Evaluated effectiveness of different chart types  

### ✅ Sales Dataset (County-level)  
- Created PivotTables summarising sales by **County** and **Product**  
- Used `SWITCH` function to categorise sales volume:  
  - **High** (>600)  
  - **Medium** (300–600)  
  - **Low** (<300)  

---

## 📊 Key Insights  
- Certain countries consistently perform across multiple markets  
- Customer demographics significantly impact sales performance  
- Clear visualisation choices improve interpretability of data  
- Categorisation logic (e.g. SWITCH) enhances analysis readability
